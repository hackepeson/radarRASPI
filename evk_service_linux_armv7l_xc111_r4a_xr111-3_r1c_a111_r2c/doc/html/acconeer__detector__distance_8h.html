Detect reflections in the provided data.

Uses the threshold provided by threshold estimation or, if no threshold estimation has been performed, the default threshold value for detecting reflections. Provides how many reflections that has been detected, calling acc_detector_distance_get_reflections provides the content of each refleciton for last detection.

Note that this function may alter the input data.

Parameters
[in]detectorA reference to the detector
[in]distance_start_mThe distance of the first data point[m]
[in]distance_end_mThe distance of the last data point[m]
[in]data_sizeThe size of data
[in]dataThe data to perform detection on, may be altered
[out]reflection_countThe number of detected reflections are provided in this parameter
Returns
Status

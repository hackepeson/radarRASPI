acc_os_mutex_t acc_os_mutex_create ( void  )

Initialize a mutex.

If a pointer to a mutex is given, that mutex is initialized. If NULL is given, a new mutex is dynamically allocated and initialized.

If the pointer to an already initialized mutex is given, nothing is done and the mutex is left in its current state. Care must therefore be taken that the mutex variable is zeroed before being passed to acc_os_mutex_init() so it can be detected to be initialized or not.

Returns
Newly initialized mutex
